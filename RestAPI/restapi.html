🏪 Yeh REST API Kya Cheez Hai?

REST API: Ek tarah ka web service jo URL ke zariye data deta hai, aur HTTP methods se control hota hai. Simple, na?

🛍️ Dukaan wala Example

🛒
GET: Dukaan se samaan mangna — "/orders" se order ki list mil jayegi!
➕
POST: Naya order daal dena — ekdum fresh order banega!
✏️
PUT: Purane order mein thodi si tabdeeli — jaise order update karna!
❌
DELETE: Kharaab order hatao — order cancel kar do!
Upar ke buttons dabao aur REST methods ka maza lo!
