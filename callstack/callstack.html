Function Definitions

// 4. secondFunction() completes
// POP: secondFunction
// 5. firstFunction() completes
// POP: firstFunction

üéØ Key Concept: LIFO Stack Behavior
PUSH: When a function is called, it's added to the top of the stack
POP: When a function completes, it's removed from the top of the stack
The call stack operates on Last In, First Out principle
LIFO: Last In, First Out
